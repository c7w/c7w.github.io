编译单元与类空间

  • 编译单元：一个 Java 源代码文件
    • 一个编译单元中只能有 一个 public 权限的类
    • 且该类名与文件名相同
    • 类名存在冲突问题
  • 编译单元的组成部分
    • 所属包的声明；
    • Import 包的声明用来导入外部类；
    • 类和接口的声明.
